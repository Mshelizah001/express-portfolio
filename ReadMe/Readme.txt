Express Portfolio Website

Author: Jesse Daniel Mshelizah

Overview

This project is my personal portfolio website built with ExpressJS, NodeJS, and EJS.
It includes four main pages — Home, About Me, Projects, and Contact Me — and was created using the Express Generator with the --view=ejs option.
The site follows the required structure with organized folders for routes, views, and public assets, and is fully responsive using Bootstrap and Font Awesome.

Features

ExpressJS server with EJS templating engine

Routes for Home, About, Projects, and Contact pages

Shared layout file (layout.ejs) with:

A Bootstrap navigation bar

A main content section (<%- body %>)

A footer with active social links
